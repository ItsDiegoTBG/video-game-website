# video-game-website

A videogame selling website that will probably me made using React, Typescript, Java and MySQL.

Project will need to have to following functionalities to be consider complete:

1. Workable Frontend and Backend (Will need to have an API.)
2. An Admin and User system. 
3. An Inventory System.
4. Favorite or Rated Games System.
5. A Cart System (Ill see if it should be a full functionality Or just removing money from a users account and add a button that will give the user money)
6. Must use JWTs
7. Must follow Solid Principales if able.
